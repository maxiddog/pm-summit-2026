PM Principles

Principle Description Status

PM Success Framework

Clear Communication

  • Write crisp PRDs that articulate the problem and proposed solution
  • Ensure alignment across all stakeholders before development begins
  • Maintain clear documentation in Confluence and JIRA

Strategic Thinking

  • Balance short-term wins with long-term vision
  • Every feature should ladder up to company OKRs
  • Prioritize ruthlessly using RICE or similar frameworks

Bias for Action

  • Move quickly, make decisions with 70% certainty
  • Course-correct based on feedback and data
  • Don't let perfect be the enemy of good
