Core Pillars

Customer Obsession

Every decision starts with the customer. We deeply understand their pain points, workflows, and goals.

Ship Fast, Learn Faster

We believe in rapid iteration and learning from real-world usage. Perfect is the enemy of shipped.

Data-Driven Decisions

We eat our own dog food. Every feature is measured, analyzed, and optimized based on real usage data.

Cross-Functional Excellence

Great products are built by great teams. We collaborate closely with engineering, design, and GTM.

Debugging Challenge

Your Mission

This swag store has 3 intentional bugs that you need to fix using the Datadog MCP server before you can complete your order.

The Bugs

  • Bug #1: Checkout button doesn't work
  • Bug #2: Selecting size "S" causes a crash
  • Bug #3: Email validation is broken

How to Debug

Use your AI assistant (Cursor or Claude Code) with the Datadog MCP server to:

  1. Query Datadog logs for errors
  2. Identify the root cause
  3. Fix the code
  4. Redeploy and test

Tip: Start by adding items to your cart and trying to checkout. The first bug will reveal itself immediately.
