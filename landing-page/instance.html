üìö Setup Instructions

  1. Clone the Repository

    Clone the swag store repo to your local machine:
    git clone https://github.com/YOUR_ORG/datadog-swag-game.git && cd datadog-swag-game/swag-store

  2. Deploy Your Instance

    Deploy your own copy of the buggy store to Vercel:
    npm install && vercel --prod

  3. Add MCP Configuration to Cursor/Claude Code

    In your editor settings, find "MCP Servers" section. Paste the JSON configuration from above. Save and restart to activate the Datadog MCP server.

  4. Open Your Store & Start Debugging!

    Visit your deployed store URL, try to checkout, and encounter the bugs. Ask your AI assistant to help debug using Datadog MCP. Fix the code locally, run vercel --prod again, and your fixes go live!
